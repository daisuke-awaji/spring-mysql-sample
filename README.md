# spring-mysql-sample

SpringBootのアプリケーションをMySQLに接続するサンプルコードです。

以下のSpringガイドから参照しています。
https://spring.io/guides/gs/accessing-data-mysql/

### GitHubからソースコードをクローン
```
$ git clone https://github.com/daisuke-awaji/spring-mysql-sample.git
$ cd spring-mysql-sample/
```

### Springアプリケーションをビルド
```
$ ./gradlew build
```

### SpringBoot 実行
```
$ sh app.jar
```
